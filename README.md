Tableau Visual Analytics: Sales Executive Dashboard
Este repositorio alberga un proyecto de visualización de datos desarrollado en Tableau, diseñado para proporcionar una visión clara y dinámica del rendimiento comercial. El dashboard transforma datos de ventas en una herramienta de exploración interactiva para la toma de decisiones.

Descripción del Dashboard
El archivo dashboar ventas.twb ofrece un análisis detallado del ciclo de ingresos, permitiendo a los usuarios filtrar por múltiples dimensiones y descubrir tendencias ocultas en los datos.

Características Principales:
Análisis Temporal: Evolución de ventas y beneficios a lo largo del tiempo.

Geolocalización: Mapas interactivos para identificar mercados clave y zonas de bajo rendimiento.

Desglose por Categorías: Comparativa de éxito entre diferentes líneas de productos.

KPIs Dinámicos: Indicadores clave de rendimiento que se actualizan automáticamente según los filtros aplicados.

Habilidades Técnicas Aplicadas
Cálculos de Tableau: Creación de campos calculados complejos para obtener métricas personalizadas (Profit Margin, YoY Growth, etc.).

Parámetros y Conjuntos (Sets): Implementación de parámetros para permitir que el usuario cambie las métricas de visualización dinámicamente.

Dashboard Action & Navigation: Uso de acciones de filtrado y resaltado para crear una experiencia de usuario fluida.

Diseño Visual (Visual Best Practices): Aplicación de principios de diseño de datos, uso de colores estratégicos y jerarquía visual para evitar la carga cognitiva.
